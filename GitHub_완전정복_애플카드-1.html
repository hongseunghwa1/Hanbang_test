Sign up for GitHub · 갤럭시 크롬 기준

GitHub란?

내가 만든 HTML 파일을 인터넷에 저장하고 웹페이지로 만들 수 있는 무료 서비스입니다. 카카오톡 클라우드처럼 파일을 보관하고 전 세계에 공개할 수 있어요.

STEP 1 · 크롬으로 접속
1
크롬(Chrome) 앱 실행
스마트폰 홈화면에서 크롬 앱을 누릅니다
2
주소창 누르기 → 아래 주소 입력
github.com
3
키보드에서 이동 또는 Enter 누르기
STEP 2 · 회원가입 진행
1
Sign up 버튼 누르기
또는 Sign up for GitHub 버튼
2
이메일 주소 입력 → Continue
예시: sample@gmail.com
3
비밀번호 설정 → Continue
영어+숫자+특수문자, 8자 이상 / 예: Kim2026!
4
사용자 이름(아이디) 입력 → Continue
예: kimai2026 · myfirstweb01 (이미 있으면 숫자 추가)
5
이메일 수신 여부 선택
n 또는 No 선택 후 Continue
STEP 3 · 퍼즐 인증
⚠️
퍼즐을 틀려도 괜찮아요!
Try again을 눌러 다시 천천히 풀면 됩니다. 누구나 한두 번 틀립니다.
퍼즐 유형진행 방법
유형 1 · 같은 그림기준 그림과 같은 그림 → Submit
유형 2 · 방향 맞추기화살표로 그림 돌리기 → Submit
유형 3 · 특정 그림해당 그림 모두 누르기 → Verify
STEP 4 · 이메일 인증
1
Gmail 앱 열기
퍼즐 인증 후 GitHub에서 이메일이 옵니다
2
제목: Verify your email address 메일 찾기
3
Verify email address 버튼 누르기
로그인 상태가 되면 가입 완료!

새로고침 → 스팸함 → 프로모션함 → 이메일 주소 재확인

AI에게 코드 추천받기

Claude AI로 깃허브 검색어 추천받기

깃허브에는 수백만 개의 코드가 있어서 처음엔 어떻게 검색해야 할지 모릅니다. Claude AI에게 원하는 것을 말하면 딱 맞는 검색어를 알려줍니다!

이렇게 질문하세요
💬 Claude에게 보내는 질문 예시
🎮 게임 찾을 때
"깃허브에서 모바일에서 실행되는 html 게임 검색어 추천해줘"
🌐 웹페이지 찾을 때
"깃허브에서 식당 랜딩페이지 html 단일파일 검색어 추천해줘"
🎨 포트폴리오
"포트폴리오 만들 수 있는 html 단일파일 깃허브 검색어 알려줘"
깃허브 직접 검색
1
github.com → 상단 검색창에 검색어 입력
2
검색 결과에서 원하는 저장소 선택
3
초록색 Code ▼ 버튼 → Download ZIP

바이브코딩으로 코드 수정

AI에게 말로 수정 요청 · 코딩 몰라도 OK!

✨
바이브코딩이란?
AI에게 말로 수정을 부탁해서 코드를 바꾸는 방법입니다. 코딩을 전혀 몰라도 됩니다!
전체 흐름
코드 복사

수정 요청
코드 복사
→
Spck 전체교체
→
저장→▶확인
STEP 1 · Spck에서 코드 복사
1
Spck 앱 → index.html 파일 열기
2
코드 화면 길게 누르기 → 전체 선택 → 복사
STEP 2 · Claude AI에게 수정 요청
3
Claude 앱 또는 claude.ai 접속
4
입력창 길게 누르기 → 붙여넣기 → 수정 내용 입력 → 전송
수정 요청 예시
💬 이렇게 말하면 됩니다
📞 전화번호 변경
"전화번호를 010-4620-2829로 바꿔줘"
🏷️ 상호명 변경
"상호명을 그림카페 고맙소로 바꿔줘"
🎨 배경색 변경
"배경색을 연한 파란색으로 바꿔줘"
🍵 메뉴 추가
"메뉴에 아메리카노 5,000원 추가해줘"
📍 주소 변경
"주소를 진해구 중원로 40-1로 바꿔줘"
STEP 3 · 수정 코드 적용
5
AI가 준 코드 박스 복사 버튼 누르기
6
Spck → 전체 선택 → 삭제 → 붙여넣기
7
저장(💾) → ▶ 버튼으로 미리보기 확인!
💡
꿀팁
한 번에 하나씩 요청하면 더 정확합니다! 마음에 안 들면 "다시 해줘" 라고 하면 됩니다.
